### 👨‍💻 Sobre mim

Sou um Engenheiro de Software apaixonado por tecnologia, com amplo domínio do ciclo de vida de desenvolvimento, desde a arquitetura de infraestrutura na nuvem até a entrega de interfaces web e mobile de alto desempenho.

- 🏢 **Atual:** Co-Founder e Desenvolvedor Full Stack na **Not Dev**, além de ser o Founder da **Stock Tracking**.
- ⏳ **Trajetória:** Atuei por mais de 4 anos como Software Engineer na HOS Sistemas, adquirindo sólida experiência na resolução de problemas complexos de software corporativo.
- 🎓 **Formação:** Bacharelado em Engenharia de Software (Estácio) e Ciência da Computação (UCS).
- 🎯 **Foco técnico:** Arquitetura de aplicações robustas e modernas utilizando React, Next.js, Node.js, Bancos de Dados Relacionais (Oracle, PostgreSQL, MySQL) e ferramentas de infraestrutura (AWS, Redis).

### 🚀 Projetos em Destaque (Vercel Deployments)

- **Rodosuper Desktop:** Um robusto sistema de gestão com backend desenvolvido em Next.js e Node.js utilizando Prisma ORM, Sequelize, banco de dados MySQL e cache em Redis. Todo ecossistema frontend operando com o ecossistema React moderno (SWR, Recharts, Tailwind CSS).
- **Laço Velho:** Solução web completa. O frontend foi construído de forma fluida usando Vite, React 19, TailwindCSS v4, Framer Motion e TanStack Query. O back-end dedicado gerencia as requisições servindo sob Next.js e integrando-se via Prisma.
- **Stock Tracking & Not Dev:** Projetos empreendedores fundados por mim, focados na resolução prática de necessidades do mercado e validação de MVPs em produção contínua.
